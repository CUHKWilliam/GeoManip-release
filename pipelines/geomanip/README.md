# GeoManip: Geometric Constraints as General Interfaces for Robot Manipulation

This is the improved version of the paper [GeoManip: Geometric Constraints as General Interfaces for Robot Manipulation](https://arxiv.org/abs/2501.09783). It replaces the geometry parser with a more efficient few-shot segmentation method, which avoid extensive VLM query and can be run in real-time.
